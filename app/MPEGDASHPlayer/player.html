Description

This example contains source code for an MPEG-DASH player using the Shaka Player package from the Shaka Player project. It will play MPEG-DASH single and adaptive bitrate vod, live & nDVR streams.

Warning: You may experience inconsistent playback using this third-party beta DASH test player.

Installation

In the /examples/VideoOnDemandStreaming directory:

  • LINUX
    Run ./installall.sh
  • WINDOWS 7/8/8.1/10
    Right-click installall.bat and then select Run as administrator
  • WINDOWS SERVER
    Double-click the installall.bat file
  • OS X
    Double-click the installall.command file

Instructions

  1. Read the Tutorials listed below.
  2. Click the Connect button. The video will start playing.

To play your own content

  1. Copy your video files to the Wowza Streaming Engine [install-dir]\content directory.
  2. In the Stream field, change the mp4 file name to match one of your video files.
  3. Click the Connect button. Your video will start playing.
